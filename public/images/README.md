# 画像ファイルの配置について

このディレクトリには、サイトで使用する画像を配置します。

## ⚠️ 著作権について（必読）

**ポケモンカードゲームの公式画像・カード画像・パッケージアートは、株式会社ポケモン／任天堂／GAME FREAK／Creatures Inc. の著作物です。**

以下のいずれかで運用してください：

### ✅ 安全な方法
1. **自分で撮影した写真** を使用する（パックを購入して撮影）
2. **公式サイトへのリンク** で済ませ、画像は配置しない
3. **引用要件を満たした上で** 公式画像を使用（出典明記・必要最小限・主従関係）

### ❌ リスクがある方法
- 公式画像をダウンロードしてそのまま自サイトにホスティング
- 他サイトから画像を転載

## ディレクトリ構成

```
public/images/
├── packs/          # 拡張パックBOX画像（アイキャッチ等）
│   └── ninja-spinner-box.svg    ← プレースホルダー
├── cards/          # カード画像
│   └── mega-greninja-mur.svg    ← プレースホルダー
└── default-thumb.svg
```

## 現在の状態

すべてのカード／BOX画像は **「画像準備中」と明示したプレースホルダーSVG** が配置されています。
実写画像に差し替える際は、同じファイル名で上書きしてください。

## 自分で撮影した画像を配置する方法

### 1. カード画像を配置

```bash
# カードを撮影した JPG を配置
cp ~/Downloads/mega-greninja-mur.jpg public/images/cards/mega-greninja-mur.jpg
```

### 2. 記事のメタデータを更新

`content/articles/mega-greninja-ex-mur-price.mdx` の frontmatter:

```yaml
---
cardImage: "/images/cards/mega-greninja-mur.jpg"   # .svg → .jpg に変更
eyecatch: "/images/packs/ninja-spinner-box.jpg"
---
```

### 3. Git にコミットしてデプロイ

```bash
git add public/images/
git commit -m "images: 実写画像を追加"
git push
```

Vercel が自動で再デプロイします。

## 推奨サイズ

| 用途 | サイズ | 備考 |
|------|-------|------|
| OGP / アイキャッチ | 1200 × 630 px | SNSシェア用に最適 |
| カード画像 | 734 × 1024 px 程度 | カード縦横比 5:7 |
| 記事サムネイル | 800 × 450 px | 16:9 比率 |
| パック商品画像 | 正方形 600 × 600 px | 商品写真向け |

## ファイル形式

| 形式 | 用途 |
|------|-----|
| **WebP** | Web配信で最も軽量。推奨 |
| **JPG** | 写真向け。汎用性高 |
| **PNG** | 透過が必要な場合 |
| **SVG** | ロゴ・図形（プレースホルダーとしても利用中） |

## 出典明記のテンプレート

記事内で画像を表示する場合は、以下のように出典を記載することを推奨：

```
出典: 株式会社ポケモン「ポケモンカードゲーム公式ホームページ」より
https://www.pokemon-card.com/ex/m4/
```
